# 2. 블랜더 단축키 정리

---

## 이동

`마우스 중간 휠` : 상하좌우 돌리기

`shift+마우스 중간 휠` : 화면 위치 이동

`마우스 휠` : 크게/작게

## side 기능

`Tab` : 편집 모드

`shift+s` → `cursor to selected` → `object` → `set origin` → `origin to 3D cursor` : 피벗 위치 변경

`r` : 로테이트

`a` : 모두 선택

`r+x` : x축 방향으로 돌리기

`n` : 정보 보기

`shift+D` : 오브젝트 복사

`shift+R` : 했던 동작 반복

## 도형

`shift+A` : 도형 추가

`e` : 도형 세로 늘리기/줄이기

`e+z` : 도형 z축으로 늘리기/줄이기

`alt+s` : 도형 가로 늘리기/줄이기

`s` : 도형 굵기 늘리기/줄이기
